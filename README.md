# homebrew-gitignore-in

Homebrew Installation of [gitignore.in](https://github.com/gitignore-in/gitignore-in).

## Installation

```console
$ brew tap gitignore-in/gitignore-in
$ brew install gitignore-in
```

## Update

```console
$ ./generate.sh
```

You can also pin the upstream release explicitly:

```console
$ ./generate.sh v0.2.0
```

To rehearse the release preparation workflow without opening a PR, run
`prepare formula update` with `mode=dry-run`.
